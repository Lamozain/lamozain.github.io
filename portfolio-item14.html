CODING to learn

Learn Coding Now (School Arme)

After learning through the first set, the programme was greatly improved with the introduction of other performance indicators such as baseline and endline tests. Twelve grade 3 pupils participated in this set. We also learned the basics such as powering on a computer, using the mouse and keyboard, shutting down the computer, and Programming BB-8.

This set then moved on to using scratch. We practiced lauching the app, choosing and resizing sprites and backdrops. We also wrote animations and cartoons. Finally we designed a game to show what we have learnt. This second set spent the same time as the first. Some also took Bebras Challenge to demonstrate their skills.

One thing I have realized is that the knowledge of coding comes with a host of 21st century skills that every learner needs. Such skills involve collaboration, problem-solving, critical thinking, computational-thinking, divergent- thinking among others. I therefore decided to train teachers too so that they can use the method in teaching their students. Hence, the School Arme project.
